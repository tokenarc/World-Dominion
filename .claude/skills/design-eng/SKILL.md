# Instructions

Follow design principles from emilkowal.ski articles:

1. **Typography:** Use proper type systems, font pairing, modular scales
2. **Color:** Strategic color usage, avoid generic palettes
3. **Layout:** Create visual hierarchy with spacing and grids
4. **Motion:** Purposeful animations with proper easing
5. **Interaction:** Focus states, loading patterns, forms

Avoid common mistakes: overused fonts, generic gradients, nested cards, gray text on colored backgrounds.
