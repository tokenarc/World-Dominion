# Instructions
1. **Analysis:** Review the git diff to understand what actually changed.
2. **Message Generation:** Create a "Conventional Commits" message (e.g., feat: add nation selection logic).
3. **Execution:** Suggest the full command: git add . && git commit -m "..." && git push.
4. **Note:** Always ask for confirmation before pushing.
